## Campus Group Platform

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/07061a9d7b8b44509f89cbe1c185ebe3/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Event Manager, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Announcements
5. Group Activities
6. Personal Center
7. Group Activity Favorites
8. Group Activity Registration
9. Backend Management

**Event Manager Functions:**
1. Login
2. Change Password
3. Personal Information
4. Announcement Management
5. Group Activity Management
6. Group Activity Message Management
7. Group Activity Registration Management



**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Basic Data Management (Class Management, College Management, Major Management, Announcement Type Management, Group Activity Type Management)
4. Announcement Management
5. Group Activity Management, Group Activity Message Management, Group Activity Favorites Management, Group Activity Registration Management
6. User Management
7. Event Manager Management
8. Carousel Information
